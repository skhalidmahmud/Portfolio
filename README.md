# MD Khalid Mahmud - Professional Developer Portfolio

A high-performance, visually stunning, and highly interactive professional portfolio website designed for a modern Software Engineer specializing in Full-Stack Development, AI, and LLMs.

## 🚀 Key Features

### 🖥️ Advanced Interactive UI
- **Interactive Developer Terminal:** A fully functional terminal emulator (`khalidOS`) where visitors can type commands like `help`, `skills`, `about`, and even discovery easter eggs.
- **Command Palette (`Ctrl + K`):** A power-user search and navigation menu for instant access to any part of the site, theme toggles, or CV downloads.
- **3D Reactive Hero Portrait:** A mouse-tracking 3D tilt and slide effect on the profile portrait with dynamic lighting and glow effects.
- **Cyberpunk Targeting Cursor:** A custom, high-FPS crosshair cursor that dynamically matches the current theme's accent color.

### 🎨 Personalization & Aesthetics
- **Premium Glassmorphism Design:** A modern, sleek aesthetic with vibrant gradients, blurred backgrounds, and smooth micro-animations.
- **Theme Switcher:** Seamless Dark and Light mode toggle with persistent storage (remembers your choice).
- **Accent Color Picker:** 6 curated premium color palettes (Blue, Emerald, Pink, Amber, Purple, Ruby) that update the entire site's theme instantly.
- **Dynamic Mesh Background:** Animated background blobs and low-opacity aesthetic textures for a premium feel.

### 🌍 Global Reach
- **Multi-Language Support:** Integrated one-click translation for 12+ major languages (English, Bengali, Hindi, Japanese, Korean, Arabic, Chinese, etc.).
- **Social Media Integration:** Fully branded social icons (Facebook, X, WhatsApp, LinkedIn, GitHub, Instagram) with hover effects.

### 💼 Professional Showcase
- **Expandable Certifications Gallery:** A clean grid showcasing 15+ professional certifications with an "Open Full Gallery" reveal feature.
- **Responsive Experience Timeline:** Detailed education and professional history presented in a modern, easy-to-read layout.
- **SEO Optimized:** Implemented meta tags, semantic HTML5, and performance best practices.

## 🛠️ Built With
- **Vanilla HTML5 & CSS3:** No heavy frameworks for ultra-fast loading times.
- **Vanilla JavaScript:** Custom logic for 3D effects, terminal logic, and command palette.
- **FontAwesome 6.5.1:** High-quality icons for all interfaces.
- **Google Fonts (Outfit):** Modern typography for better readability.

## 📱 Mobile Responsive
The site is fully optimized for all screen sizes, from high-resolution desktop monitors to mobile devices, ensuring a consistent premium experience.
